Inclure du JS dans une page HTML

Directement dans le code HTML :

<script>alert('Hello world!');</script>

Dans un fichier .js :

index.html

<script type="text/javascript" src="script.js" defer></script>

script.js

alert('Hello world!');
Il est préférable de positionner la balise <script> avant la fermeture de la balise <body> L'attribut defer permet d'attendre que le DOM soit chargé avant d'exécuter le code JS

Les fonctions alert / prompt / confirm

  • alert() afficher un message à l’internaute
  • prompt() demander à l’internaute d’effectuer une saisie
  • confirm() demander à l’internaute de confirmer une action
alert("Hello world!");
prompt("Saisissez votre pseudo", "pjehan");
confirm("Etes-vous sûr ?");
En JS, une ligne peut se terminer par un ";" ou bien en retour à la ligne

L'objet console

L'objet console permet d’afficher des informations de debug dans la console :

console.log('log');
console.debug('debug');
console.info('info');
console.warn('warn');
console.error('error');
