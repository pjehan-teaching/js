Manipuler le DOM

Qu'est-ce que le DOM ?

Le DOM (Document Object Model) est une interface de programmation pour les documents HTML ou XML. Il représente la structure d'un document et définit la façon dont cette structure est accessible par un programme qui pourra ensuite en modifier la structure, le style ou le contenu.

Le DOM est une représentation d'un document sous forme de nodes et d'objets ayant des propriétés et des méthodes.

Créer un élément dans le DOM

L'objet document permet construire de nouveaux éléments et de les ajouter sur la page HTML :

  • createElement() : Créer un nouvel élément HTML
  • createTextNode() : Créer une nouvelle zone de texte
  • appendChild() : Ajouter un élément enfant
  • removeChild() : Supprimer un élément enfant
let title = document.createElement('h1');
title.innerText = 'Hello world!';
document.querySelector('body').appendChild(title);

Modifier les attributs

L'objet element permet de modifier les attributs :

  • setAttribute() : Définir la valeur d’un attribut
  • getAttribute() : Récupérer la valeur d’un attribut
document.querySelector('input').setAttribute('value', 'test');

Modifier le style

Pour modifier le style d'un élément, il est préférable d'utiliser la propriété style :

document.querySelector('input').style.borderColor = 'red';

Vous pouvez également ajouter des classes CSS en utilisant la méthode classList.add() :

document.querySelector('input').classList.add('is-invalid');
