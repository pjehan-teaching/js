Les variables

Présentation des variables

Les variables sont comme des conteneurs qui permettent de stocker des informations en mémoire.

En JavaScript, les variables ne sont pas typées mais elles ont tout de même un type implicite.

Il est possible d'utiliser les mots clés var ou let pour déclarer des variables :

var firstName = 'John';
let lastName = 'Doe';
Le mot clé let à été introduit avec l'ES6

var VS let

La différence entre les mots clés var et let se trouve dans leur portée (ou scope en anglais).

  • var : Portée globale (Global Scope)
  • let : Portée locale (Block Scope)
{
    var firstName = 'John';
    let lastName = 'Doe';
}

console.log(firstName); // John
console.log(lastName); // Uncaught ReferenceError: lastName is not defined
Il est préférable d'utiliser let Une variable déclarée dans une fonction n'est pas accessible en dehors de celle-ci

Les constantes

Avec ES6 est également apparu le mot clé const permettant de déclarer une constante.

Contrairement aux variables, les constantes ne peuvent pas changer de valeur :

const PI = 3.14;
PI = 3; // Uncaught TypeError: invalid assignment to const 'pi'
Par convention, le nom des variables s'écrit en majuscules

Nommage des variables

  • Le nom des variables doit être composé uniquement de caractères alphanumériques (A-z, 0-9, & et _ )
  • Le nom des variables ne doit pas contenir d’accents (utiliser un nommage en anglais ?)
  • Le nom des variables ne doit pas commencer par un chiffre
  • Le nom des variables est sensible à la casse (case-sensitive).
  • Ainsi var hello est différente de var Hello
  • Lors de la déclaration, les variables ne sont pas typées
  • On utilisera généralement la syntaxe camelCase pour le nommage des variables

Les types de variable

En Javascript, une même variable peut contenir des informations de différents types :

Type Description
Chaîne de caractères (string) Texte écrit entre guillemets ou apostrophes
Nombre entier (int) Nombre entier positif ou négatif
Nombre à virgule (float) Nombre à virgule positif ou négatif (la virgule s’écrit avec le caractère “.”)
Booléens (bool) Valeur vraie (true) ou fausse (false)
Vide (undefined / null) Aucun type, variable
Il est possible de tester le type d’une variable avec l’instruction typeof

Affecter une valeur à une variable

let nom = 'Dupont'; // Variable de type string
let prenom = "Jean"; // Variable de type string
let message = 'J\'ai compris le Javascript'; // Variable de type string
let age = 22; // Variable de type int
let taille = 1.80; // Variable de type float
let homme = true; // Variable de type bool
let profession = null; // Variable vide

La concaténation

let nom = 'Dupont'; // Variable de type string
let prenom = "Jean"; // Variable de type string
let message = 'Bonjour ' + nom + ' '  + prenom;

⚠️ Le signe "+" étant utilisé pour la concaténation, l’opération ne donne pas forcément le résultat attendu.

console.log("1" + 1);

Calculs

Il est possible avec le langage JavaScript d'effectuer des opération arithmétiques basiques :

Opérateur Description
+ Addition
- Soustraction
/ Division
* Multiplication
% Modulo (reste de la division)

Calculs

Cet exemple permet de calculer la différence entre deux notes :

let note1 = 12;
let note2 = 15;
alert('Différence entre les deux notes : ' + (note2 - note1));

Incrémentation

En programmation, il est souvent nécessaire d'incrémenter une valeur numérique.

Cela peut se faire de différentes manières :

let valeur = 10;
valeur = valeur + 1; // valeur = 11
valeur++; // valeur = 12
valeur = valeur + 2; // valeur = 14
valeur += 2; // valeur = 16

L'objet Math

Pour réaliser des opérations mathématiques plus avancées, JavaScript dispose d'un objet Math comprenant entre autres ces méthodes :

  • Math.PI : Récupérer la valeur de Pi
  • Math.round() : Récupérer l'entier le plus proche
  • Math.random() : Retourne une valeur aléatoire entre 0 et 1
