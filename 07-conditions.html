Les conditions

Introduction

Les conditions permettent d'exécuter une ou plusieurs instructions en fonction du résultat d’une comparaison.

Exemple : Si le nombre de produits en stock est inférieur à 10, alors il faut se réapprovisionner. Sinon, si le nombre de produits est supérieur à la place disponible en stock, alors il faut les mettre en avant dans le magasin.

Introduction

On pourrait représenter l'algorithme précédent avec le code Javascript suivant :

let nombreDeProduits = 8;
let tailleMaxStock = 30;

if(nombreDeProduits < 10) {
  nombreDeProduits = nombreDeProduits + 10;
} else if (nombreDeProduits < tailleMaxStock && nombreDeProduits > (tailleMaxStock - 5)) {
  alert('Attention, nous allons bientôt avoir trop de produits en stock.');
} else if (nombreDeProduits > tailleMaxStock ) {
  alert('Il faut mettre ce produit en avant !');
}

Les symboles de comparaison

Symbole Description
== Est égal à
> Est supérieur à
>= Est supérieur ou égal à
< Est inférieur à
<= Est inférieur ou égal à
!= Est différent de
Les symboles === et !== permettent également de comparer le type de donnée

Combinaison de conditions

let nombreDeProduits = 2;
let tailleMaxStock = 30;
let jourSemaine = 'Dimanche';
if(nombreDeProduits < 10 && jourSemaine != 'Dimanche') {
  nombreDeProduits = nombreDeProduits + 10;
} else if (nombreDeProduits > tailleMaxStock) {
  alert('Mettre ce produit en avant !');
} else {
  alert('Rien à signaler.');
}

Les booléens

Avec les variables de type booléen, on peut utiliser une syntaxe simplifiée :

let isAdmin = true;

if (isAdmin == true) { }
if (isAdmin == false) { }

// Syntaxe simplifiée
if (isAdmin) { }
if (!isAdmin) { } // Ici, le point d'exclamation permet d'inverser la condition

Ternaire

Il existe également une autre façon d'écrire des conditions appelée condition ternaire :

message = (score > 10) ? "Vous avez gagné" : "Vous avez perdu";
