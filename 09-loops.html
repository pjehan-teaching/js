Les boucles

Introduction

Les boucles permettent de répéter une série d'instructions en fonction d’une condition.

Tant que la condition est respectée, on exécute la série d'instructions.

const countries = ['France', 'Germany', 'Belgium'];

for (i = 0; i < countries.length; i++) {
    console.log(countries[i]);
}

La boucle while

La boucle while permet d'effectuer une itération tant que la condition est vraie

let compteur = 3;

while(compteur > 0) {
  console.log(compteur);
  compteur--;
}

console.log('Bonne année !!');

La boucle for

La boucle for est similaire à la boucle while mais elle permet d'instancier, d'initialiser et de redéfinir la valeur d’une variable plus facilement :

for(let pourcentage = 0; pourcentage < 100; pourcentage++) {
    console.log('Chargement en cours. ' + pourcentage + '% effectué...');
}
La boucle for permet de ne pas oublier l’incrémentation de la variable

La boucle forEach

En JavaScript, il existe également une méthode forEach() que l'on peut utiliser sur les tableaux.

Cette méthode prend en paramètre une fonction qui sera appelée pour chaque élément du tableau :

const countries = ['France', 'Germany', 'Belgium'];

countries.forEach(function(country) {
    console.log(country);
});

La boucle for of

Cette boucle également de boucler sur les éléments d'un tableau :

const countries = ['France', 'Germany', 'Belgium'];

for (let country of countries) {
    console.log(country);
}

La boucle for in

Cette boucle ne permet pas de boucler sur un tableau mais sur les propriétés d'un objet :

const person = { firstName: 'John', lastName: 'Doe', salary: 2500 };

for (let property in person) {
    console.log(property); // firstName ... lastName ... salary
}
