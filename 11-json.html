Le format JSON

Introduction

JSON (JavaScript Object Notation) est un format permettant de stocker et d’échanger des données.

Il n’est pas lié à un langage de programmation en particulier :

// JavaScript
JSON.stringify(data);
// PHP
json_encode(data);
// C#
System.Web.Helpers.Json.Encode(data);
// Java
new JSONObject(data);

Contraintes

Les données au format JSON doivent respecter les contraintes suivantes :

  • Les données sont stockées sous forme de clé / valeur
  • Les données sont séparées par une virgule
  • Les accolades permettent de contenir un objet
  • Les crochets permettent de contenir un tableau

Exemple

{
    "glossary": {
        "title": "example glossary",
        "GlossDiv": {
            "title": "S",
            "GlossList": {
                "GlossEntry": {
                    "ID": "SGML",
                    "GlossTerm": "Standard Generalized Markup Language",
                    "Acronym": "SGML",
                    "GlossDef": {
                        "para": "A meta-markup language, used to create markup languages such as DocBook.",
                        "GlossSeeAlso": ["GML", "XML"]
                    },
                    "GlossSee": "markup"
                }
            }
        }
    }
}
L'autre format souvent utilisé est XML
